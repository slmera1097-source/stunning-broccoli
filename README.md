# Nibble

**Track calories, simply.**

Nibble is a fast, lightweight calorie tracking app that makes logging meals effortless. Snap a photo, scan a barcode, or quick-add common foods in seconds.

## Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Bun + SQLite
- **Build:** Vite

## Getting Started

```bash
bun install
bun run dev
```

The app will be available at `http://localhost:3000` (Vite) with the API server on `http://localhost:3001`.

## Features

- Manual food logging with calorie tracking
- Barcode scanning (Pro)
- Photo logging (Pro)
- Daily and weekly dashboards
- Macro and micronutrient tracking (Pro)
- Custom calorie goals
- Data export (Pro)
